# AgentHelm Examples

This directory contains working examples demonstrating AgentHelm features.

## Examples

### 1. Basic Agent (`basic_agent.py`)

Simple ToolAgent with custom tools.

```bash
python examples/basic_agent.py
```

### 2. Planning Workflow (`planning_workflow.py`)

Generate and execute multi-step plans.

```bash
python examples/planning_workflow.py
```

### 3. Multi-Agent Orchestration (`multi_agent.py`)

Multiple agents working together with dependencies.

```bash
python examples/multi_agent.py
```

### 4. MCP Integration (`mcp_example.py`)

Connect to MCP servers and use their tools.

```bash
python examples/mcp_example.py
```

## Prerequisites

```bash
pip install agenthelm
export MISTRAL_API_KEY="your-key"
```
